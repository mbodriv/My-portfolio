# Maria Bodero — Portfolio

Personal portfolio built with React, showcasing my background across finance, data and full-stack development.

🌐 **Live site:** https://mbrportfolio.netlify.app
---

## Tech Stack

- React 19 + Vite
- Framer Motion
- React Type Animation
- React Icons
- QRCode.react
- CSS custom properties (dark/light mode)

---

## Features

- Single-page layout with sidebar navigation
- Dark/light mode toggle
- Typing animation on hero section
- Skills tab system
- Project cards with live and GitHub links
- QR code contact card

---

## Getting Started

```bash
git clone https://github.com/mbodriv/my-portfolio
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)
